## Get started in three steps

<Steps>
  <Step title="Install the package">
    Install GoodBoyDoc from PyPI using pip:

    ```bash
    pip install goodboydoc
    ```
  </Step>
  <Step title="Generate documentation from source">
    Pass your Python source to `generate` and choose a docstring style:

    ```python
    from goodboydoc import generate, format_output

    source = open("mymodule.py").read()
    docs = generate(source, style="google")
    print(format_output(docs, fmt="markdown"))
    ```
  </Step>
  <Step title="Configure for your project (optional)">
    Create a `.goodboydoc.json` file in your project root to set persistent defaults:

    ```json .goodboydoc.json
    {
        "style": "google",
        "output_format": "markdown",
        "include_private": false,
        "max_line_length": 79
    }
    ```
  </Step>
</Steps>
